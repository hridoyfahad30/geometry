DOM is Document Object Model. The purpose of DOM is dynamically manipulate a website using javascript.

The methods of selecting a HTML Element by using DOM is written below:

1. document.getElementById()
2. document.getElementsByClassName()
3. document.getElementsByTagName()
4. document.querySelector()
5. document.querySelectorAll()


DOM methods are:-

1. HTML DOM getElementById() Method.
2. HTML DOM getElementsByClassName() Method.
3. HTML DOM getElementsByName() Method.
4. HTML DOM getElementsByTagName() Method.
5. HTML DOM querySelector() Method.

Event Bubble is a function of DOM which, when clicked on any element or any part of the website, moves up from that place step by step towards the parent element.

Call back function is a function which is declared somewhere and can be called elsewhere according to our need and we can also call it as a parameter inside another function if we want.
